# Equipment Replace Mod

A [MelonLoader](https://melonwiki.xyz/) mod for MDRG that adds a Replace button to each row in the Equipment Sets UI.

## What it does

The vanilla Equipment Sets UI only lets you equip a saved set or export it. This mod adds a Replace button next to each Export button that overwrites that set with your current equipment and clothing, without changing its name or position in the list.

A confirmation dialog appears before any changes are made.

## Requirements

- MelonLoader installed on your game

## Installation

1. Download `EquipmentReplaceMod.dll` from the [Releases](../../releases) page
2. Place it in your game's `Mods/` folder
3. Launch the game

## Instructions

1. Open the Equipment Sets menu
2. Click Replace on any set
3. Confirm the prompt
4. The set is updated with your current equipment
